# Pokedex

You can check this out right away going to: https://pokedex-aajuarezd.web.app/
## Stack

- React as main framework
- React-Query for async calls
- Tailwind for styling
- DaisyUI for theming
- Firebase for hosting

## How to run

1. Clone the repo
2. run `npm install`
3. run `npm start`
